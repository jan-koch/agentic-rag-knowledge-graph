Agent Name Override Test

Test 1: Widget with Custom Agent Name

This widget should display "My Custom Support Bot" in the header:

Note: Replace 'your-api-key-here' with your actual API key to test. 
<script>
  window.RAG_CHAT_CONFIG = {
    apiKey: 'your-api-key-here',
    apiUrl: 'https://botapi.kobra-dataworks.de',
    agentName: 'My Custom Support Bot',
    language: 'en'
  };
</script>
<script src="https://botapi.kobra-dataworks.de/static/chat-widget-secure.js"></script>

How to Test

  1. Copy one of the code snippets above
  2. Replace 'your-api-key-here' with your actual API key
  3. Paste the code into a new HTML file
  4. Open the HTML file in a browser
  5. Check the agent name displayed in the widget header

Available Configuration Options

window.RAG_CHAT_CONFIG = {
  apiKey: 'required',           // Your workspace API key
  apiUrl: 'optional',            // Default: 'https://botapi.kobra-dataworks.de'
  agentName: 'optional',         // Override agent name from database
  greeting: 'optional',          // Custom greeting message
  language: 'optional',          // 'en' or 'de' (default: 'de')
  primaryColor: 'optional',      // Custom primary color (default: '#667eea')
  position: 'optional'           // 'bottom-right', 'bottom-left', etc.
};
